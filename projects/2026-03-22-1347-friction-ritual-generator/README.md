# Friction Ritual Generator

A playful browser tool for turning avoided tasks into tiny ceremonial attack plans.

## What it does

You type in the thing you're resisting, then choose:

- dread level
- time budget
- noise tolerance
- desired energy
- reward style
- witness mode

The app generates:

- a ritual name
- a short framing summary
- an opening move
- a micro-goal
- scene-setting guidance
- a payoff / reward
- anti-sabotage rules
- a step-by-step ritual script
- three alternate mood variants

## Why it exists

A lot of procrastination is just theatrical friction.
This project turns that friction into structure with a little bit of pageantry.

## Run it

Open `index.html` directly in a browser.
